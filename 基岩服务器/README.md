# Minecraft基岩服务器一键开服
> 最初的版本来自于大佬James A. Chambers，https://github.com/TheRemote/MinecraftBedrockServer
> 我在他的基础上进行本地化二次开发，之后的内容和功能可能与他的不再一致，请注意。

这是一个MC基岩服务器的一键开服工具，我尽可能地减少开设一个Minecraft基岩服务器所需的步骤。

## 怎么做？
在你的用户根目录下执行以下命令：
```shell
wget https://raw.githubusercontent.com/TheRemote/MinecraftBedrockServer/master/SetupMinecraft.sh<br>
chmod +x SetupMinecraft.sh<br>
./SetupMinecraft.sh<br>
```
